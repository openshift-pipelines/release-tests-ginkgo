  <name>Task 1: Port pkg/triggers/ from reference repo with Gomega assertions</name>
  <files>pkg/triggers/triggers.go, pkg/triggers/helper.go</files>
  <action>
Create `pkg/triggers/` directory and port two files from `reference-repo/release-tests/pkg/triggers/`:

**triggers.go** -- Port all functions with these changes:
1. Change package import paths from `github.com/openshift-pipelines/release-tests/pkg/*` to `github.com/openshift-pipelines/release-tests-ginkgo/pkg/*`
2. Remove ALL `github.com/getgauge-contrib/gauge-go/gauge` and `github.com/getgauge-contrib/gauge-go/testsuit` imports
3. Replace `testsuit.T.Errorf(...)` with `Expect(...).To(...)` or `Fail(...)` from Gomega -- import `. "github.com/onsi/gomega"` and `"github.com/onsi/ginkgo/v2"`
4. Replace `testsuit.T.Fail(err)` with `Expect(err).NotTo(HaveOccurred())`
5. Replace `gauge.WriteMessage(...)` with `GinkgoWriter.Printf(...)`
6. In `MockPostEvent()`: Remove `gauge.GetScenarioStore()["payload"] = eventBodyJSON` -- instead, the function should return BOTH the response and the payload bytes as a second return value: `func MockPostEvent(...) (*http.Response, []byte)`. The caller will hold the payload in a local variable.
7. In `MockPostEvent()`: The call to `buildHeaders()` currently reads payload from store via `store.GetPayload()`. Change `buildHeaders()` to accept the payload as a parameter instead: `func buildHeaders(req *http.Request, interceptor, eventType string, payload []byte) *http.Request`
8. In `AssertElResponse()`: Replace `gauge.GetScenarioStore()` usage with direct parameter passing. The function already takes all needed params.
9. In `CleanupTriggers()`: Replace `testsuit.T.Fail(err)` with `Expect(err).NotTo(HaveOccurred())`. Remove the hardcoded `os.Getenv("GOPATH")+"/src/github.com/..."` path for cert cleanup -- use `config.Path("testdata/triggers/certs")` instead (matching what the rest of the codebase uses).
10. Keep `time.Sleep` calls as-is for initial migration (noted for future Eventually replacement).

**helper.go** -- Port with these changes:
1. Same import path changes as above
2. Remove `testsuit` and `store` imports
3. `GetSignature()`: Replace `testsuit.T.Errorf` with `Expect(err).NotTo(HaveOccurred())`
4. `buildHeaders()`: Change signature to accept `payload []byte` parameter instead of calling `store.GetPayload()`. Use the payload parameter for HMAC signature calculation.
5. Replace `testsuit.T.Errorf` in default case of `buildHeaders()` with `Fail("unsupported interceptor type: " + interceptor)`

Ensure `go build ./pkg/triggers/` compiles cleanly.
